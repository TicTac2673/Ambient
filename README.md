## how to use?

1. clone da repo:

    ```bash
    git clone https://github.com/ImLegiitXD/Ambient.git  
    cd Ambient
    ```

 2. build it using gradle:

    ```bash
    ./gradlew build
    ```
    or if ur using windows
    ```bash
    gradlew.bat build
    ```

 4. how to run:

    open a terminal window and run this command
    ```bash
    java -cp "client.jar;assets.jar" Start
    ```
